
# adas-team-website
The official Ada's Team website.

## Our Tech Stack
- [Jekyll](https://jekyllrb.com/docs/)
- [Bulma](https://bulma.io/documentation/)

## To run this website on your local machine
`bundle exec jekyll serve --livereload`
